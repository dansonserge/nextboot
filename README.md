####
Use this project to quickly start your project with React NextJs, Bootstrap, Sass, Fontawesome.

### How to start develop
```

git clone git@github.com/dansonserge/nextboot.git
cd nextboot
yarn
npm run dev
```

### How to export project
```
npm run build
npm run export

```

### Install Prerequest
```
npm install -g serve
```

### See static output
```
cd out
serve -p 8080
```
